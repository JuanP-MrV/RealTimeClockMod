## ğŸ® Usage Guide
- **Press `P`** to enter the **preview mode**.
- Use **`AWSD` or arrow keys** to reposition the clock.
- Press **`P` again** to **save the position**.
- Press **`Esc`** to **cancel** and exit preview mode without saving.
